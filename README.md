# fws02

## Read Google Forms Output

1. Put output from GForms (in `.zip` form) into `/input`
2. `cd` to `pdir`
3. run `p0_unzip.sh`
4. run `p1_process.sh`
5. output will be at `/output/OS231_GH.txt` 

## Quiz R/W
1. Put the input into `input/` as `i02.txt`
2. `cd` to `pdir`
3. run `p2.sh`
3. output will be at `/output/o2.txt`
